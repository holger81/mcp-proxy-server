MCP Proxy

Config: /data/config/servers.json, domains: /data/config/domains.json. Each server belongs to a domain (unique id + label) for the public MCP surface: clients only see searchToolsForDomain (query or explicit listAll, paginated), searchTool, and callTool — not every upstream tool at once. Stdio installs use PyPI/npm under /data/venvs or /data/npm.
